## Code Solution

```java
class Solution {
    static ArrayList<Long> factorialNumbers(long N){
        ArrayList<Long> result=new ArrayList();
        long factorial=1;
        for(long i=1;i<=N;i++){
            factorial*=i;
            if(factorial<=N) result.add(factorial);
            else break;
        }
        return result;
    }
}
```
